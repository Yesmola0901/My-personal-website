# Personal Website (OOI YEE GUEN)

This is a personal website created for the frontend development assignment. It is built using HTML, CSS, and JavaScript, featuring a responsive design and a 3D flipable photo gallery.

## Features
- **Frontend Only**: No backend or database used.
- **Responsive Design**: Works on desktop and mobile.
- **Multilingual Support**: English and Traditional Chinese versions.
- **Interactive Elements**: 3D photo card, animations, and hover effects.
